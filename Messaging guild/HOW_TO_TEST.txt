================================================================================
                         CÁC TEST USERS CÓ SẶN
================================================================================

Không cần login, dùng trực tiếp:

  ✅ user1    → Gửi tin cho user2 hoặc user3
  ✅ user2    → Gửi tin cho user1 hoặc user3
  ✅ user3    → Gửi tin cho user1 hoặc user2
  ✅ alice    → Gửi tin cho bob
  ✅ bob      → Gửi tin cho alice

EXAMPLE:
  {"from":"user1","to":"user2","content":"hello"}  ✅ OK
  {"from":"alice","to":"bob","content":"hi"}       ✅ OK
  {"from":"user1","to":"alice","content":"msg"}    ✅ OK

================================================================================
                           6 API ENDPOINTS
================================================================================

1. GỬI TIN NHẮN
   POST /message/send
   Input:  {"from":"user1","to":"user2","content":"hello"}
   Output: Message with ID, timestamp, sender, content

2. XEM INBOX
   GET /message/inbox/{userId}
   Input:  /message/inbox/user2
   Output: List of messages received by user2

3. XEM CONVERSATION
   GET /message/conversation/{user1}/{user2}
   Input:  /message/conversation/user1/user2
   Output: All messages between user1 & user2

4. MARK AS READ
   POST /message/read/{userId}/{messageId}
   Input:  /message/read/user2/abc123
   Output: Success response

5. DELETE MESSAGE
   DELETE /message/{userId}/{messageId}
   Input:  /message/user2/abc123
   Output: Success response

6. CHECK STATUS (ONLINE)
   GET /message/status/{userId}
   Input:  /message/status/alice
   Output: {userId, isOnline: true/false}

# TEST GỬIV NHẮN TỪ user1 ĐẾN user2

$body = '{"from":"user1","to":"user2","content":"Hello from user1"}'
$response = Invoke-WebRequest -Uri "http://localhost:8080/message/send" `
  -Method POST `
  -ContentType "application/json" `
  -Body $body `
  -UseBasicParsing

Write-Host $response.Content

# TEST XEM INBOX CỦA user2

  -UseBasicParsing

# TEST CONVERSATION

$conv = Invoke-WebRequest -Uri "http://localhost:8080/message/conversation/user1/user2" `
  -Method GET `

================================================================================
                        TROUBLESHOOTING
================================================================================

❌ "Address already in use" - Port 8080 đang dùng
  → Cách 1: Tắt cái kia, chạy lại
  → Cách 2: Dùng port khác: .\mvnw.cmd spring-boot:run -Dspring-boot.run.arguments="--server.port=8081"

❌ "Could not connect" - App chưa chạy
  → Chạy: .\mvnw.cmd spring-boot:run
  → Chờ "Tomcat started on port 8080"

❌ "400 Bad Request" - Sai input
  → Kiểm tra JSON format
  → Kiểm tra "from" và "to" là users có sẵn
  → Kiểm tra "content" không trống

❌ "404 Not Found" - Endpoint sai
  → Kiểm tra URL: /message/send, /message/inbox, etc.
  → Kiểm tra method: POST vs GET
